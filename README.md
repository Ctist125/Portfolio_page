# Tief's Portfolio

- 위 프로젝트(Project)는 개인 포트폴리오 페이지(Portfolio page)입니다.
- Node.js와 Express를 사용해 서버 측 코드를 구축하였고, MongoDB를 데이터베이스로 사용했습니다.

## Skills

### Frontend

HTML, CSS, JavaScript

### Backend

Node.js, Express

### Database

MongoDB
